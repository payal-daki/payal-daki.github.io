01 — ABOUT

Infrastructure is a product. I run it like one.

I'm a DevOps Engineer at Mplify Tech Services, where I've been the sole engineer responsible for cloud infrastructure since July 2024 — managing everything from provisioning to deployment to incident response across nine-plus live products.

My work spans migrating applications to AWS from the ground up, building CI/CD pipelines that only rebuild what's changed, running live database migrations without losing a row, and keeping deployments boring — in the good way. I hold an AWS Solutions Architect Associate certification and a B.Tech in Computer Science & Engineering with a focus in Cyber Security.

Recently I've been extending that foundation into Kubernetes and Terraform, moving from hand-managed EC2 deployments toward infrastructure that's declarative, versioned, and easier to scale.
